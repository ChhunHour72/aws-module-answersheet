1.Cloud definition
+ Cloud computing is the on demand delivery of computer power, database, storage ,application and other resourse on the internet with pay as you go pricing.

2.Cloud service models
+ There are 3 cloud service models
- IaaS(Infrasrructure as a service) 
- PaaS(Platform as a service)
- SaaS(Software as a service)

3.Cloud deployemet models
+ There are 3 cloud deployment models
- Cloud : all part of application run in the cloud.
- Hybrid : connect existing infrastructure and applications to a cloud-based resources.
- On-premises(private cloud) : use virtualization and resources management tool sometimes called private cloud.

4.Benefit of cloud computing
+ The 6 main advanages of cloud computing are 
- Trade for capital exspense for variable cost : Very cost effictive.
Note : Capital expense or capex are the cost that you pay for where you use it or not.
Variable expense pay only when you use and the amount you consume.
- Massive economies of scale : AWS can achive higher economies and pass saving to customers. This mean lower pay as you go for comsumer.
- Stop guessing capacity : you can access as much or as little as you need.
- Increase speed and agility : time it take to experiment and develop are lower.
- Stop spending money on running and maintaining data center.
- Go gobal in minutes : deploy applications in minutes.
